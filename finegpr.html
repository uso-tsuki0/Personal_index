Research of Construction and verification of synthetic datasets based on GTA-V Game Engine

Overview

GOAL

Develop a generator which could generate multi-module synthetic datasets with GTA-V game engine and scripting tools.

Result

  • A DLL file used as plugin for GTA-V
  • A python file for post-processing

Duration

It's a haf-year research program from Sep 2022 to Mar 2023.

Abstract

Pedestrian Re-identification (REID) refers to a computer vision task to identify pedestrians from various sources, which have been widely applied to security monitoring and intelligent transportation. However, current datasets are plagued with a lack of scale, resolution and annotations, which can be an obstacle to effective model training. Inspired by previous project FineGPR, which has used GTA-V for modeling and data-generation, our study aims to provide a more diverse, flexible and detailed dataset for REID model training on the basis of FineGPR. Using the GTA-V game engine as a simulation platform as well as ScriptHookV.Net as script runner, we generate the dataset. With full-screen RGB images from the FineGPR dataset as the foundation, we introduce skeletal point annotations similar to the CrowdPose, changes in camera height, depth map information, automatic figure boundary estimate as well as movement labels. The generated dataset surpasses the existing ones with its improved scale, environmental consistency, fine resolution, multidimensional information and detailed annotation. It provides a valuable resource for training supervised or semi-supervised REID models. Our study demonstrates the advantage of using existing game engine for generating large-scale, multi-module, and finely annotated datasets, pointing out the perspective direction of training dataset preparation. Future researches are needed to expand the dataset while exploring other applications of gaming engines in computer vision tasks.

Keywords: REID, GTA-V, game engine, data set, script

Background

Pedestrian Re-identification (REID) refers to a computer vision task to identify pedestrians from various sources, which have been widely applied to security monitoring and intelligent transportation. However, the existing datasets for REID suffer from limitations in scale, resolution, and annotations, hindering effective model training. In this proposal, we aim to address this challenge based on the success of the FineGPR project, which employed the GTA-V game engine for data generation. Our study aims to create a diverse, flexible, and detailed dataset for REID model training, surpassing the limitations of current datasets.

The current research has introduced various datasets, such as Market-1501[1], CUHK03[2], DukeMTMC-reID[3], etc. These datasets are collected in real-world environments. However, such datasets have certain limitations, such as a small number of samples, high manual annotation costs, and privacy concerns. To address these issues, the article [4] proposes the use of the Grand Theft Auto V (GTA-V) game engine to generate the person re-identification dataset GPR-800. Compared to previous real datasets, GPR-800 has significant advantages: free sampling and annotation, large scale, diverse scenes, and high resolution [4]. However, GPR-800 also has room for improvement; its text labels only provide information such as time, environment, and character names, and the images in the dataset differ from those encountered in reality. For example, in terms of camera angles, the camera angles in GPR-800 are all horizontal, unlike the tilted angles of cameras in reality.

Methodology

Tools

  • Grand Theft Auto V (GTA-V)
  • ScriptHookV.Net
  • Native DB

Camera position

I expanded the degree of freedom of the camera, enabling it to cover a hemisphere above the character.

  • radius: 1m
  • Angle difference: pi/4
  • Use (theta, phi) to represent a point
  • 17 positions in total

Skeleton point

Skeleton points' positions were obtained through scripts and beam method was applied to judge occlusion. If the beam goes across another instance, there is an occlusion.
